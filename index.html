About ScrutaNet

ScrutaNet is an open-source multi-threaded distributed password cracking system. It includes server GUI, server CLI, and client CLI for distributed hash-cracking tasks.

Features

  • GUI version (Qt6) with light/dark theme
  • CLI server / client for lightweight use
  • Multi-threaded, supports wordlist mutation, etc.
